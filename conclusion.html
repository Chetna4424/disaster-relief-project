Conclusion

The project successfully demonstrates how algorithms like Dijkstra’s and Priority Queue Sorting can optimize disaster relief operations.

  • Quick identification of shortest and most efficient delivery routes.
  • Urgency-based sorting ensures timely aid distribution.
  • Visualization and simulation improve decision-making accuracy.

Future Work

  • Integrate GPS and real-time traffic data.
  • Automate decision-making using AI prediction models.
  • Expand the system to handle multiple relief centers.

This system lays the foundation for scalable, AI-assisted disaster response management.
