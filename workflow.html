System Workflow

The workflow of the Flexible Disaster Relief Allocation System includes several key steps to ensure efficient and prioritized resource distribution.

  1. Collect input data – relief center, disaster areas, and distances.
  2. Assign urgency levels to each disaster area.
  3. Apply Dijkstra’s Algorithm to compute shortest paths.
  4. Use Priority Queue + Sorting to determine delivery sequence.
  5. Simulate the delivery process (route traversal and completion).
